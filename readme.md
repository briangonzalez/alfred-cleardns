# alfred-cleardns

> Clear DNS via Alfred.


## Install

```
$ npm install --global alfred-cleardns
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `cleardns`, then hit <kbd>Enter</kbd>.
